Thorax Classification - Diaz Dwi Kurniawan - F1B019040

1. Dataset berjumlah total 7.200 data (4000 Normal, 700 TBC, 2500 Pneumonia).
2. Link Dataset Penelitian : https://drive.google.com/drive/folders/1ccQWDaJtSEaGojvfv8kVgapnGsauwK0-?usp=sharing
3. Dataset sudah di Split dengan perbandingan 80:10:10 (80% train, 10% validation, 10% test).
4. Total data train : 5760 (3200 normal, 560 tbc, 2000 pneumonia).
5. Total data validation : 720 (400 normal, 70 tbc, 250 pneumonia).
6. Total data test : 720 (400 normal, 70 tbc, 250 pneumonia).

7. Source code di run dengan Google Collab menggunakan GPU.
8. File "streamlit-thorax-classification" merupakan source code python implementasi model kedalam aplikasi dengan framework Streamlit.
